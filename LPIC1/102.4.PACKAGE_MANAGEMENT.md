# PACKAGE MANAGEMENT

Package or app downloading was apperared for the first time on Linux (father of AppStore, Google Play, ...).

debian packages' name: NAME-VERSION-RELEASE_ARCH.deb

Where is the config that say to `apt` where should the system can download repos:

```bash
cd /etc/apt/
less sources.list
```

`sources.lists.d` directory is for our custom repos for updating.

`apt update` or `apt-get update`: Goes to all sources and updates list of all the programs. Puts it in a cache: `/var/cache/apt/pkgcache.bin`
