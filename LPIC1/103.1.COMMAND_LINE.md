# SHELL

Terminal Emulator is a program that runs Shell!

BASH: Bourne Again SHell

Absolute Path / Relative Path

Special Chars (like \*, $, ...).

# CMDs

```bash
echo $SHELL

cd # empty cd takes you into the home! (like ~)

type cd

pwd

uname # system name
uname -s # kernel name
uname -n # hostname
uname -r # kernel release
uname -v # kernel version
uname -m # cpu name
uname -a # all of the above

man

touch x\ y # to create a file with 'x y' name
```
